# Talent Squad Demo

This is a React + Vite demo project for **Talent Squad Europe**.  
It showcases the platform's hero section, step-by-step process, and platform overview components.

---

## Features

- Animated hero section with three phases on scroll
- Step-by-step process with images
- Platform overview cards with SVG icons
- Responsive layout
- Tailwind CSS for styling

---

## Technologies

- React 19
- Vite
- Tailwind CSS
- Swiper (for sliders if needed)
